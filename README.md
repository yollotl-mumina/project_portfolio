# Portafolio · Daniela Zúñiga

Portafolio de investigación (trilingüe ES/EN/FR). Las fotos van en la carpeta `img/`, así el `index.html` es ligero (~60 KB) y carga rápido.

## Estructura del repositorio
```
index.html
img/                      (15 fotos: hero, AgroParisTech, Campeche, comunidades)
CV_Daniela_Zuniga.pdf
Reporte_NH3_Grignon.pdf
Informe_Diversidad_Fungica_Sascaberas.pdf
README.md
```

## Publicar en GitHub Pages (recomendado: subir archivos, no pegar)
1. Crea el repositorio `yollotl-mumina.github.io` (o `portafolio`).
2. **Add file → Upload files** y arrastra el `index.html`, la carpeta `img/` y los 3 PDF.
   - En computadora puedes arrastrar la carpeta `img/` completa y GitHub sube todo respetando la estructura.
3. **Settings → Pages → Deploy from a branch → `main` → `/root` → Save.**
4. En 1–2 min abre `https://yollotl-mumina.github.io/`.
